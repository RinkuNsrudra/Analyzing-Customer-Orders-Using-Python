Project Overview
This project focuses on analyzing customer order data using Python's built-in data structures and control flow techniques. The objective is to identify customer purchasing patterns, classify customers based on spending behavior, analyze product category performance, and generate actionable business insights for an e-commerce company.

The project demonstrates how Python lists, tuples, dictionaries, sets, loops, and conditional statements can be used to process and analyze customer order information efficiently without relying on external data analysis libraries.

Business Problem
As a Data Analyst for an e-commerce company, the goal is to analyze customer purchase data across multiple product categories such as Electronics, Clothing, and Home Essentials. Business managers require insights into:

Most frequently purchased products

High-value customers

Customer spending patterns

Popular product categories

Revenue contribution by category

Inventory and marketing optimization opportunities

The analysis helps stakeholders make data-driven decisions regarding sales strategies, customer retention, and inventory management.
